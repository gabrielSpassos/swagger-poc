# Person Service - Swagger Poc - Spring Boot

### Usage

* Clone the repository 

* Run the jar

* Access this url: 

```
http://localhost:9000/swagger-ui.html
```

#### Person Service Endpoints
```
[Method: GET]
http://localhost:9000/people

[Method: GET]
http://localhost:9000/people/[person id]

[Method: DELETE]
http://localhost:9000/people/[person id]
```
